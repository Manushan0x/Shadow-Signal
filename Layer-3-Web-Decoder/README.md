# Spy Agent — WAV Morse Decoder

A browser-based decoder for the `output.wav` files generated by the **SpyAgent Java Swing project**.

## How to Use

1. Open `index.html` in any modern browser (Chrome, Firefox, Edge)
2. Run your Java `Spy_Agent` project to generate `output.wav`
3. Drag & drop `output.wav` onto the decoder, or click to select it
4. Watch the signal processing pipeline decode your message!

## Signal Processing Pipeline

The decoder is calibrated to match your Java project exactly:

| Step | Details |
|------|---------|
| WAV Parser | Reads PCM bytes, supports 8-bit and 16-bit WAV |
| 800 Hz Bandpass | IIR filter isolating morse carrier from ambient noise |
| Envelope Detection | Full-wave rectify + 15ms moving-average smooth |
| Morse Detection | Uses exact DOT=90ms, DASH=250ms, GAP=80ms from AudioGenerator.java |
| Morse → Text | Same character map as MorseConverter.java |

## Matched Java Parameters

```
AudioGenerator.FREQ  = 800 Hz  (morse carrier)
AudioGenerator.DOT   = 90 ms
AudioGenerator.DASH  = 250 ms
AudioGenerator.GAP   = 80 ms
AudioMixer:          signal×2 + noise÷2
AmbientMusic:        220Hz + 330Hz + 110Hz sine + Gaussian noise
```
